# PassportAI 🌍

> Your personalized immigration navigator. One sentence. 8 seconds. Your entire roadmap.

Built at the McGill Data Network Hackathon 2026 — Theme: Automate. Optimize. Innovate.

## What it does
Type your background in plain language. PassportAI builds your complete Life Passport:
- Visa & legal action items with deadlines
- Matched scholarships for your field and city
- Local community organizations from your home country
- Career roadmap tailored to your profession in Canada

## Discover Feed
A TikTok-style scroll of "Did you know?" immigration facts about countries you may never have considered.

## Tech Stack
React · Vite · Tailwind CSS · Node.js · Express · Google Gemini AI

## Run locally
```bash
git clone https://github.com/YOURUSERNAME/passportai
cd passportai
npm install
echo "GEMINI_API_KEY=your_key" > .env
npm run dev
```
